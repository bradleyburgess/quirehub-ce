Welcome to QuireHub

QuireHub is an all-in-one choir management solution that aims to centralize all aspects of running a choir.

This is the Community Edition of QuireHub, and the project is still under active development.

There will also in the future be a paid, hosted version of QuireHub, which will have additional features.

If you're seeing this page, it's likely because you are running QuireHub for the first time. This page will guide you through the (very short) setup process.
